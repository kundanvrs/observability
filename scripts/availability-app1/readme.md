**** =========================
# How to Run
# =========================
# 1. Install dependencies
#    pip install -r requirements.txt
#
# 2. Start monitoring (in one terminal)
#    python monitor.py
#
# 3. Start dashboard (in another terminal)
#    python app.py
#
# 4. Open browser
#    http://localhost:5000/uptime
#    http://localhost:5000/logs
